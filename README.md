## 👨🏻‍💻 Installation

This app requires [Node.js](https://nodejs.org/) v10+ to run.

```
npm install //to install requiered packages
```

And then run:

```
npm start //to run the app!
```

🥳 Vala, You will see the webbrowser poppin up!
